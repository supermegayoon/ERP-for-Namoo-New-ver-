# ERP Namoo V7

## 포함 기능
- 로그인 및 역할별 메뉴
- Style / 주문 / 생산 / 재고 / 거래처 / 공급업체
- Buyer 원가 Excel 다중 시트 Import
- 같은 Style 재업로드 시 원가 버전 V1, V2, V3 이력 보존
- 원자재 / 부자재 / 공정 / 기타비용 직접 추가
- 최초 견적 / 1차 네고 / 최종 원가 비교
- 제조원가 VAT-/VAT+ 자동 계산
- 최신 원가가 매출·수익으로 자동 연결
- Selling FOB 입력 시 Unit Profit / Margin / Markup / 총매출 / 총이익 계산
- Dashboard KPI 및 저마진 Style 표시
- 원가 및 매출·마진 Excel Export
- 카드형 UI로 좌우 스크롤 최소화
